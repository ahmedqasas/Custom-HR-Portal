# Custom HR Portal (Single-Page Application) 🏢

A fully custom, responsive Human Resources portal built natively within Odoo 19, designed to bypass standard system views and deliver a highly optimized, modern user experience.

### 🎯 The Challenge
Standard ERP views, while functional, can sometimes be rigid for specific, employee-facing interfaces. The objective was to build a fluid, modern HR dashboard that breaks away from the traditional ERP look and feel, ensuring high adoption rates and ease of use for all employees.

### 💡 The Solution
I architected a Custom Single-Page Application (SPA) directly within the Odoo environment. By bypassing the standard XML-based system views, I established a custom file structure and implemented tailored widget layouts optimized perfectly for both mobile and desktop configurations.

### 🛠️ Tech Stack & Tools
* **ERP Engine:** Odoo 19
* **Frontend Framework:** OWL (Odoo Web Library)
* **Styling:** Tailwind CSS
* **Architecture:** Single-Page Application (SPA)

### ✨ Key Technical Highlights
* **Custom Widget Layouts:** Designed specific UI components from scratch to display relevant HR data clearly without the constraints of standard views.
* **True Responsiveness:** Seamless, native-feeling experience across desktop and mobile devices leveraging Tailwind CSS utilities.
* **Optimized Performance:** The SPA architecture ensures lightning-fast load times and smooth transitions between HR modules.
